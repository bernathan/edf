Overview ▹

Overview ▾

This package aims to implement an interface to the EDF+ standard as described on http://www.edfplus.info/ for the Go language. The objective is to read a file and provide some kind of access to the stored data stored. It was written by Cristiano Silva Jr. while working at the Laboratory of Neuroscience and Behavior from the University of Brasilia and was released under the GPL license.
